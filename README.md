# CIFAR-10 Fusion: ResNet50 + MobileNetV2

## Overview
Mini case study for Graphics & Visual Computing: feature-level fusion of ResNet50 and MobileNetV2 on CIFAR-10.

## How to run (Google Colab)
1. Open `notebooks/run_all.ipynb` in Colab.
2. Runtime → Change runtime type → GPU.
3. Run all cells. Figures will be saved to `report/figs/`.

## Reproducibility
- Random seed used: `42`.
- Hyperparameters and training logs are in the notebook.
